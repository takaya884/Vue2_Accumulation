# Vue 2 Option API 勉強用プロジェクト

Vue.js 2系のOption APIを学習するためのシンプルなサンプルアプリケーションです。

## 🚀 機能

- **データバインディング**: 基本的なリアクティブデータの表示
- **双方向データバインディング**: v-modelによる入力フォームの連携
- **条件付きレンダリング**: v-if/v-elseによる動的な表示切り替え
- **リストレンダリング**: v-forを使ったTODOリスト機能
- **算出プロパティ**: computedによる動的な値の計算
- **ウォッチャー**: watchによるデータ変更の監視
- **ライフサイクルフック**: created, mounted, updatedの実装例

## 📁 プロジェクト構成

```
vue2-option-api-study/
├── index.html           # メインのHTMLファイル（Vue CDN版を使用）
├── package.json         # プロジェクト設定と依存関係
├── package-lock.json    # 依存関係のロックファイル
├── node_modules/        # インストールされた依存関係
└── README.md           # プロジェクトの説明書
```

シンプルな構成で、単一のHTMLファイル内にVue 2のすべての学習内容が含まれています。

```json
  "author": "",
  "license": "MIT"

```

### 📋 各項目の解説

#### 基本情報
- **name**: プロジェクト名。npmに公開する場合の識別子
- **version**: プロジェクトのバージョン番号（セマンティックバージョニング）
- **description**: プロジェクトの説明
- **main**: メインエントリーポイント。今回は学習用なので `index.html`
- **keywords**: プロジェクトを見つけやすくするためのキーワード
- **author**: 作成者情報
- **license**: ライセンス情報（MIT = オープンソース）

#### スクリプト
- **dev**: `npm run dev` で開発サーバーを起動（ポート3000、自動でブラウザを開く）
- **serve**: `npm run serve` で開発サーバーを起動（ブラウザは手動で開く）

#### 依存関係
- **dependencies**: 本番環境でも必要なパッケージ
  - `vue@^2.7.16`: Vue.js 2系の最新版（CDNで読み込むため実際は未使用）
- **devDependencies**: 開発時のみ必要なパッケージ
  - `http-server@^14.1.1`: 静的ファイルを配信するHTTPサーバー

### 🔧 バージョン記号の意味
- `^2.7.16`: 2.7.16以上、3.0.0未満（メジャーバージョンは固定）
- `~2.7.16`: 2.7.16以上、2.8.0未満（マイナーバージョンも固定）
- `2.7.16`: 完全に固定されたバージョン

## 📦 インストール・実行方法

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動（自動でブラウザが開きます）
npm run dev

# またはブラウザを手動で開く場合
npm run serve
```

サーバーが起動すると、http://localhost:3000 でアクセスできます。

## 🎯 学習内容

### 1. データバインディング
- **テンプレート構文**: `{{ }}` を使った値の表示
- **リアクティブデータ**: `data` オプションでの状態管理
- **イベントハンドリング**: `@click` などのイベントリスナー

### 2. 双方向データバインディング
- **v-model**: 入力フォームとデータの双方向連携
- **リアルタイム更新**: 入力内容の即座な反映

### 3. 条件付きレンダリング
- **v-if / v-else**: 条件による表示切り替え
- **トランジション**: CSSアニメーションとの組み合わせ

### 4. リストレンダリング
- **v-for**: 配列データの繰り返し表示
- **TODOアプリ**: 追加・削除機能の実装
- **キーの重要性**: `:key` による効率的な更新

### 5. 算出プロパティ (Computed)
- **動的計算**: 他のデータに基づく値の自動計算
- **キャッシュ機能**: 依存データが変わらない限り再計算しない
- **パフォーマンス**: メソッドとの違いとメリット

### 6. ウォッチャー (Watch)
- **データ監視**: 特定のデータの変更を検知
- **副作用処理**: ログ出力や外部APIの呼び出し
- **深い監視**: オブジェクトの内部変更も検知可能

### 7. ライフサイクルフック
- **created()**: インスタンス作成時
- **mounted()**: DOM要素へのマウント完了時  
- **updated()**: データ更新によるDOM更新時
